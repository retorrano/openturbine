Theory
======

This section contains theoretical background on wind turbine modeling.

.. toctree::
   :maxdepth: 2

Aerodynamics
-----------

OpenTurbine uses Blade Element Momentum (BEM) theory for aerodynamic analysis.

Structural Mechanics
-------------------

Beam modeling is used for structural analysis of turbine components.

Control Systems
---------------

Pitch and torque control strategies are implemented for power optimization.
